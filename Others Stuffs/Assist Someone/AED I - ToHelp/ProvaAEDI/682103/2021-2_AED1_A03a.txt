Enunciado

Um posto de saúde pretende oferecer um turno extra para
aplicar vacinas. Haverá três unidades para aplicar cada
uma de três doses. Para isso precisará de um programa 
para organizar melhor a lista de interessados.

Uma lista manual, contendo dados recolhidos, passará 
a ser digital, mas precisará passar por filtragens para 
remoção de dados inconsistentes e/ou duplicados.

As listas filtradas por vacina (1 a 3) serão gravadas
em arquivos para uso posterior em cada unidade de 
vacinação.

Resultado

Arquivo (output_xx.txt) contendo um resumo com 
a quantidade de pessoas a serem vacinadas para cada dose, 
a partir de dados válidos de pessoas a serem vacinadas, 
apenas um valor por linha - o CPF.

Dados

Arquivo (input_xx.txt) contendo dados não filtrados
de pessoas a serem vacinadas, um valor por linha.

Condições

Verificar as seguintes condições apra a aplicação
das vacinas:

  1a. dose - sem restrições
  2a. dose - 30 dias após a primeira dose
  3a. dose - 150 dias (05 meses) após a segunda dose

Restrições

a.) Só poderão ser tratados dados válidos:
    - CPF  - valor inteiro positivo
    - data - dia no intervalo [01-30]
           - mês no intervalo [01-12]
           - ano do intervalo [2020-2022]
    OBS.: Um dado NÃO válido deverá ser representado pelo
          valor padrão (0,(0,0,0)), caso necessário.
b.) A quantidade de dados em arquivo NÃO será fornecida.
c.) Um programa modelo (A202102_03a.cpp) será fornecido com
    com as definições de dados a serem empregadas, bem como
    testes previstos. Os métodos a serem desenvolvidos já se 
    encontrarão listados e com suas definições essenciais.
d.) Tratar outras possíveis situações de erros.
